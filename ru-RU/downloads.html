† Есть два ABIs в windows используеться родная (native) (MSVC) ABI используемая в Visual Studio, и GNU ABI использующая GCC toolchain. Какую версию Rust выбрать зависит от того какие с/с++ библиотеки вы используете : для взаимодействия с библиотеками, собранными в VS, мы используем MSVC билд версию Rust. Для взаимодействия с GNU програмным обеспечением мы используем MinGW/MSYS2 toolchain use the GNU build.

MSVC билд Rust допалнительно зависит от Microsoft Visual C++ build tools for Visual Studio 2013 или позднее. Самый простой путь получения утилит сборки, является установка Microsoft Visual C++ Build Tools 2015 Другой вариант,это установить Visual Studio 2015 или Visual Studio 2013, и вовремя установки выбрать "C++ tools". Никакого дополнительного программного обеспечения не требуется для базовой сборки типа GNU.

Поддержка в Rust GNU ABI являеться более стабильной, и рекомендуется для повседневного(обычного,типичного)использования.
